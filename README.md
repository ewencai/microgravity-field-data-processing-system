# Microgravity Field Data Processing System

Micro-gravity field data processing system for CG-5/CG-6 gravimeter data processing, GPS coordinate correction, tide correction, and field survey workflow management.

## Features

- CG-5/CG-6 gravimeter data import and processing
- GPS coordinate correction (RTK)
- Tide correction
- Terrain correction
- Field survey workflow management
- Data visualization and export

## System Requirements

- Windows 10/11 (64-bit)
- MATLAB Runtime R2026a
