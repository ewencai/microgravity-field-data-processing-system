## 1. 安装与启动

### 1.1 安装
1. 双击安装包 `微重力野外数据处理软件_Setup_V1.22.2.exe`，选择安装目录。
2. 如提示安装 **MATLAB Runtime**，请按向导完成（首次安装约需数分钟）。
3. 首次启动会弹出**激活框**：将本机机器码发给软件提供方，收到签发的授权文本后粘贴激活（详见 [FAQ](#6-常见问题faq)）。
4. 完成后桌面生成快捷方式。

### 1.2 启动
- 双击桌面图标启动。程序会先校验授权，通过后进入主界面。
- 若提示"未找到授权文件 / 授权不匹配 / 已过期"，请见 [FAQ](#6-常见问题faq)。

### 1.3 首次启动说明
- 首次运行无历史记录，各参数使用**齐全的默认值**（含常用重力/GPS 仪器参数表、基点、校正标量），一般改值或取消勾选即可，无需从空表新增；之后每次关闭会自动记住路径与参数，下次自动恢复。

## 2. 界面总览与通用操作

### 2.1 界面结构
- 顶部为 **12 个功能选项卡**：重力数据、GPS数据、数据匹配、数据校正、偏移空点、加密统计、质检控制、闭合检查、进度表图、采集航迹、检查日报、成果汇总。
- 每个面板分两区：
  - **左侧参数区**：加载按钮、参数输入、复选框、信息框、运行/保存按钮。
  - **右侧图件/结果区**：多个子标签页展示图表与结果表。

### 2.2 通用按钮约定
| 按钮 | 外观 | 作用 |
|------|------|------|
| 运行 | **绿色**、白字加粗 | 执行本面板计算并绘图 |
| 保存 | 默认灰色 | 导出本面板成果到标准目录 |
| 加载 | 默认 | 选择输入数据/参数文件 |

### 2.3 路径与参数记忆
- 任一加载/保存对话框都会**定位到上次使用的目录**。
- 修改的参数（密度、阈值、带号、勾选项等）关闭软件后仍保留，下次自动填回。
- **参数统一保存**：重力参数、GPS 参数、基点与校正参数的"保存/加载"按钮均读写同一个 `项目参数.mat` 文件（各占独立数据段），无需再管理多个分散的参数文件。

### 2.3b 数据目录约定
- 重力/GPS 数据读取时**只识别"日期命名的文件夹"**（如 `2025.10.01`），项目根目录下的散落文件（参数、说明文档等）会被自动忽略。
- 因此可以放心把项目相关文件与日期数据文件夹同放在一个项目文件夹里，便于归档，不影响数据读取。

### 2.4 信息框
- 左侧信息框实时显示带时间戳的运行日志（✅ 成功 / ⚠ 警告 / ❌ 失败），可上下滚动查看。

### 2.5 窗口缩放
- 可任意拖拽窗口大小，面板与图表会自动按比例适配。

## 3. 标准作业流程

推荐按以下顺序作业（每步完成后点"保存"归档到对应目录）：

```
① 重力数据整理  ┐
                 ├─▶ ③ 数据匹配 ─▶ ④ 数据校正
② GPS 数据整理  ┘                    │
                                       ▼
  ⑤ 偏移空点 → ⑥ 加密统计 → ⑦ 质检控制 → ⑧ 闭合检查
                                       │
                                       ▼
  ⑨ 进度表图 → ⑩ 采集航迹 → ⑪ 检查日报
                                       │
                                       ▼
                          ⑫ 成果汇总（KPI + 归档 + Word/PPT + 日志）
```

> 提示：⑫ 成果汇总优先读取本次会话的内存结果；若某步未在本次运行，则自动从"预处理结果"目录已保存的成果表回退读取。

## 4. 各功能面板操作步骤

### 4.1 重力数据
1. 点【加载重力数据】，选择重力仪原始文件所在的项目**根目录**（软件会自动识别其下的日期命名文件夹，忽略散落文件）。
2. 如需调整点型关键词或阈值，在参数表中编辑（默认字典已确认，一般无需改动）。
3. 点【运行】：软件自动识别仪器类型并分类重塑，右侧显示读数热力图与时序图。
4. 点【保存】：输出到 `1.GraDataReshaped/`（原始文件 + `.xlsx` + `.txt`）。
> 参数可点【保存参】/【加载参】写入或读取统一项目参数文件 `项目参数.mat`。

### 4.2 GPS 数据
1. 点【加载 GPS 数据】，选择 GPS 原始 CSV 所在的项目根目录（自动识别日期文件夹）。
2. 在仪器类型表确认/新增仪器（E90/X10/T7，默认已预置常用仪器）。
3. 点【运行】：自动映射列布局并重塑，右侧显示点位与高程图。
4. 点【保存】：输出到 `2.GPSDataReshaped/`（保持 GBK 编码）。
> 参数可点【保存参】/【加载参】写入或读取统一项目参数文件 `项目参数.mat`。

### 4.3 数据匹配
1. 分别【加载 GRA】【加载 GPS】重塑成果。
2. 设置时间基线、时间 PC 阈值。
3. 点【匹配】：按规则匹配，右侧 LED 状态图显示匹配情况。
4. 点【保存】：输出到 `3.PointMatched/`。

### 4.4 数据校正
1. 【加载匹配数据】。
2. 设置密度、布格系数、带号；如需地形校正，勾选并【加载 DEM】。
3. 可【加载基点参数】表。
4. 点【运行】：依次完成零漂/固体潮/基点差/高度/纬度/中间层/（地形）校正，右侧分层展示散点与 13 节统计。
5. 点【保存】：输出到 `4.PointCorrected/`。

> 地改注意：DEM 与测站高程差 > 50m 提示警告，> 200m 判定地改失败（垂直基准不一致）。

### 4.5 偏移空点
1. 【加载设计点】【加载实测点】。
2. 设置放样/偏移阈值。
3. 点【运行】：统计偏移误差与空点，右侧多阈值分色散点图。
4. 点【保存】：输出到 `5.Deviation&Null/`。

### 4.6 加密统计
1. 【加载数据】（含加密点）。
2. 设置带号。
3. 点【运行】：加密点分布图与点型构成。
4. 点【保存】：输出到 `6.DensePoint/`（供地改使用）。

### 4.7 质检控制
1. 【加载质检数据】。
2. 设置高程/平面/重力质检阈值。
3. 点【运行】：误差空间分布 + 直方图 + 超限统计。
4. 点【保存】：输出到 `7.QCPoints/`。

### 4.8 闭合检查
1. 【加载闭合数据】。
2. 点【运行】：重力差/高程差/时长趋势与直方图。
3. 点【保存】：输出到 `8.ClosedUnit/`。

### 4.9 进度表图
1. 【加载】设计表、成果表、AM/PM 索引、GPS/质检表等。
2. 在阈值表设置各参数达标线。
3. 点【运行】：进度、点数柱状图、各参数趋势图。
4. 点【保存】：输出到 `9.ProgressStatistics/`。

### 4.10 采集航迹
1. 【加载】航迹数据。
2. 点【运行】：显示采集轨迹。
3. 点【保存】：输出到 `10.CollectingTracks/`。

### 4.11 检查日报
1. 【加载】日报数据（进度已运行时可自动继承）。
2. 点【运行】：生成每日检查报表与图件。
3. 点【保存】：输出到 `11.DailyCheckReport/`。

### 4.12 成果汇总〔重点〕
1. 点【加载】，选择本工区"预处理结果/工区"根目录；软件自动反推并填入工区名。
2. 确认报告版本号，按需勾选：
   - ☑ **保存时归档汇总**（复制各成果目录到汇总归档）
   - ☑ **保存时生成 Word 报告**（`.docx`）
   - ☑ **保存时生成 PPT 汇报**（`.pptx`）
3. 点【运行】：计算全工区 KPI，右侧显示汇总指标表、进度饼图、点数柱状图、质检不合格柱状图。
4. 点【保存】，选择保存目录（默认 `12.ProjectSummary/`），软件输出：
   - `ProjectSummary_KPI.xlsx`（指标表）
   - `CollectionProgress.png` / `PointStatistics.png` / `QCNonconforming.png`（图件）
   - `汇总归档/`（如勾选归档）
   - `成果汇总报告_日期.docx`（如勾选 Word 报告）
   - `成果汇报_日期.pptx`（如勾选 PPT 汇报）
   - `ProjectSummary_Log.txt`（处理日志）
5. 完成后弹窗显示保存路径。

## 5. 成果保存与报告生成

### 5.1 成果目录
所有面板成果统一保存至 `预处理结果/工区/1.~12.` 分级目录（软件自动创建）。

### 5.2 报告说明（Word / PPT）
- 报告分 **Word（.docx）** 与 **PPT（.pptx）** 两种格式，可各自独立勾选生成。
- 生成方式：软件在 MATLAB 侧构建结构化数据模型（JSON），再由内置 Python（python-docx / python-pptx）渲染成文档，文字、数字、图件、表格按既定版式精确填入。
- 当前流水线已产出的图件（进度饼图、点数统计、质检误差、闭合差等）会自动填入；暂无对应成果的图位优雅跳过，不影响报告生成。
- 生成报告需目标机具备随包分发的 Python 运行环境（含 python-docx / python-pptx），安装包已内置。

## 6. 常见问题（FAQ）

**Q1：启动提示"未找到授权文件"？**
首次使用或授权文件丢失时，软件弹出激活框。将本机机器码发给软件提供方，收到签发的授权文本后粘贴激活即可。授权文件 `License.key` 存放在 `%APPDATA%\MicroGravityLicense\`（回退至程序目录），请勿手动创建或修改。

**Q2：启动提示"授权与本机不匹配 / 已过期"？**
授权采用 RSA-2048 签名绑定机器指纹，换机或到期后软件弹出激活框。提供新机器的机器码向提供方重新申请授权文本粘贴即可。切勿回拨系统时钟试图延长授权——软件内置时钟防回拨保护，回拨后反而可能误判过期。

**Q3：GPS 数据整理后中文乱码？**
软件保持原始 GBK 编码，请用中文 Windows 的记事本/Excel 打开；请勿手动转成 UTF-8。

**Q4：地形校正提示"地改失败：垂直基准不一致"？**
DEM 与测站高程差超过 200m，请检查 DEM 高程基准是否与 GPS 一致。

**Q5：成果汇总运行后 KPI 全为 0 / "-"？**
本次未运行任一前序面板且预处理结果目录为空。请先完成前序流程或选择包含成果的工区目录。

**Q6：软件运行中断/报错？**
软件为防御式设计，一般不崩溃。若个别文件读取失败会记入信息框并跳过。可查看信息框 ⚠/❌ 提示定位问题文件；必要时重启软件（参数与路径会自动恢复）。

**Q7：换了电脑，参数/路径都没了？**
参数记忆保存在软件目录的 `app_state.mat`，随机器保存。换机需重新设置一次，之后即持久化。
