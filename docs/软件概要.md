# 微重力野外数据处理软件 · 软件概要

> 版本：V1.22.2（内部 V22）　｜　文档类型：软件概要（Software Overview）

---

## 一、软件定位

微重力野外数据处理软件是一款面向微重力野外勘探**全流程**的一体化数据处理与质量管控桌面软件。软件将重力仪原始数据整理、GPS 定位数据整理、点位匹配、多项重力校正、质量检查、进度统计到成果汇总报告生成的完整作业链条集成于单一图形界面，供野外作业与数据处理人员在**无需编程**的前提下，完成从仪器原始文件到标准成果及可交付报告的端到端处理。

## 二、开发背景

传统微重力数据处理依赖多个分散脚本，中间成果需人工传递、格式各异、易出错且难以质控。本软件通过一体化图形界面，规范作业流程、提升处理效率、强化质量管控，并实现成果的标准化归档与自动化报告输出。

## 三、总体架构

- **形态**：MATLAB App Designer 单文件类（约 1.8 万行）+ 27 个外部功能函数库；经 MATLAB Compiler 打包为 Windows 独立 `.exe` + MATLAB Runtime 运行。
- **界面**：选项卡式组织为 **12 个功能面板**，统一"左侧参数区 + 右侧图件/结果区"布局；懒加载启动、窗口自适应缩放。
- **报告**：MATLAB 侧构建结构化 JSON 数据模型，随包分发的内置 Python（python-docx / python-pptx）渲染 Word 与 PPT 双格式成果报告。
- **授权**：RSA-2048 非对称签名授权，机器指纹绑定，纯离线可验，时钟防回拨。

## 四、核心功能（12 面板）

重力数据整理 → GPS 数据整理 → 数据匹配 → 数据校正 → 偏移空点 → 加密统计 → 质检控制 → 闭合检查 → 进度表图 → 采集航迹 → 检查日报 → 成果汇总。

## 五、技术特点

- **多仪器自适应**：4 类重力仪（CG5 / CG6 / Burris-DD / Burris-JT）+ 3 类 GPS（华测 E90 / X10 / T7）自动识别，可界面扩展。
- **完整校正链**：零漂 → 固体潮 → 基点差 → 高度 → 纬度 → 中间层（布格）→ 地形（多面体正演）→ 绝对重力 / 准布格异常。
- **防御式容错**：全流程 `isfield / exist / try-catch` 守卫，缺文件/空表退默认值，全程不崩溃。
- **智能记忆**：路径与参数跨会话持久化，重启恢复作业现场。
- **标准化成果**：统一"预处理结果/工区/1.~12."分级目录，成果可追溯、可交付。
- **双格式报告**：一次建模同时驱动 Word 与 PPT，图件按需嵌入、缺图优雅跳过。

## 六、运行环境

- 操作系统：Windows 10 / 11（64 位）
- 运行支撑：MATLAB Runtime（与打包版本一致）
- 报告组件：内置 Python（python-docx / python-pptx，随包分发）
- 内存：建议 8 GB 及以上
- 授权：需配套授权文件（一机一码，启动校验）
