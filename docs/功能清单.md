# 微重力野外数据处理软件 · 功能清单

> 版本：V1.22.2（内部版本 V22 / 2026-07）　｜　文档类型：常见功能清单（Feature Checklist）

本清单按 12 个功能面板逐项列出软件能力，供功能核对、著作权登记功能点罗列、验收对照使用。

---

## 一、通用与系统级功能

| 编号 | 功能 | 说明 |
|------|------|------|
| G-01 | 多面板选项卡式界面 | 12 个功能 Tab，统一"左参数区 + 右图件/结果区"布局 |
| G-02 | 窗口自适应缩放 | 面板/坐标轴/表格随窗口按比例联动，适配不同分辨率 |
| G-03 | 路径记忆 | 各加载/保存按钮记住上次目录，下次自动定位（14+ 处） |
| G-04 | 跨会话参数持久化 | 关闭时保存、启动时恢复参数与路径（`app_state.mat`） |
| G-05 | 自动重载 | 启动时按上次路径回填数据，支持直接运行 |
| G-06 | 统一信息框 | 每面板带时间戳的运行日志滚动窗口 + 状态标签 |
| G-07 | 防御式容错 | 缺失文件/列/空表退默认值，全流程不崩溃 |
| G-08 | 标准化成果目录 | 自动创建 `预处理结果/工区/1.~12.` 分级结构 |
| G-09 | 保存完成提示 | 保存后弹窗显示成果路径 |
| G-10 | 日期文件夹智能筛选 | 重力/GPS 读取仅识别日期命名文件夹，忽略项目根目录下散落文件 |
| G-11 | 参数齐全默认预置 | 重力/GPS 仪器参数表、基点、校正标量均带完整默认值 |
| G-12 | 项目参数统一保存 | 重力参数、GPS 参数、基点与校正参数统一写入单一 `项目参数.mat` |
| G-13 | RSA 非对称签名授权 | RSA-2048 + SHA256withRSA 机器绑定、纯离线可验、时钟防回拨 |

## 二、数据整理与匹配

### 2.1 重力数据（GRA Reshape）
| 编号 | 功能 |
|------|------|
| GRA-01 | 自动识别 CG5 / CG6 / Burris-DD / Burris-JT 四种数据格式 |
| GRA-02 | 按关键词字典分类（目标/核心/扩展/加密/缺省/静态/动态/联测往返/重力基点 共 10 类点型，含普测与质检） |
| GRA-03 | CG5 数字 KeyNum、CG6/Burris 字母缩写两套识别规则 |
| GRA-04 | 结构化重塑并输出 `.xlsx`（表格）+ `.txt`（原始文本行） |
| GRA-05 | RawData 原始文件直接拷贝归档（不转码，防乱码） |
| GRA-06 | 首末点名一致性检查 |
| GRA-07 | 读数分布热力图、单位时序图可视化 |
| GRA-08 | 重力参数表 / 阈值表可编辑并持久化 |

### 2.2 GPS 数据（GPS Reshape）
| 编号 | 功能 |
|------|------|
| GPS-01 | 支持华测 E90（41 列）/ X10 / T7（42 列）三种列布局 |
| GPS-02 | 列映射结构体驱动，自动识别仪器格式 |
| GPS-03 | 用户可通过界面新增仪器类型（动态 ColMap） |
| GPS-04 | 跨仪器可选列守卫访问（Role/Base 等字段兼容） |
| GPS-05 | 多仪器结果汇总（vertcat 聚合 + 去重） |
| GPS-06 | 保持 GBK 原始编码，中文 Windows 无乱码 |
| GPS-07 | 点位分布、高程散点可视化 |

### 2.3 数据匹配（Match）
| 编号 | 功能 |
|------|------|
| MAT-01 | 按时间基线 + 测线 + 点号匹配重力与 GPS 记录 |
| MAT-02 | 基点/辅点/静态点按控制点 + 仪器匹配（Base 规则） |
| MAT-03 | LED 条带状匹配状态散点图 |
| MAT-04 | 闭合单元时序详情图（重力折线 Y 轴自适应） |
| MAT-05 | 匹配成功率状态标签实时更新 |
| MAT-06 | 匹配成果保存（含路径推导） |
